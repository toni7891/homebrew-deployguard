# Toni7891 Deployguard

## How do I install these formulae?

`brew install toni7891/deployguard/<formula>`

Or `brew tap toni7891/deployguard` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "toni7891/deployguard"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
